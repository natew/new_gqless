title: Installation
---

To install the runtime client, and the codegen CLI:

```bash yarn2npm
yarn add gqless
yarn add -D @gqless/cli
```

## Subscriptions support

```bash yarn2npm
yarn add @gqless/subscriptions
```

## React bindings

```bash yarn2npm
yarn add @gqless/react
```

## Development tools

```bash yarn2npm
yarn add @gqless/logger
```
